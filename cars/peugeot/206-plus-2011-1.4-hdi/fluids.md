# Fluids

## Engine oil

| Field | Value |
|---|---|
| Danish | Motorolie |
| English | Engine oil |
| Specification | 5W-30 with PSA B71 2290 approval |
| Buy amount | 5 L |
| Fill amount | Approx. 3.8 L incl. oil filter, but fill gradually and verify on dipstick |
| Recommended | TotalEnergies Quartz Ineo ECS 5W-30 |
| Alternatives | Mobil 1 ESP 5W-30, Castrol C2/C3 5W-30 with PSA approval |
| Status | Oil spec should be checked against owner manual/service data before bulk buying |

Practical method: fill about 3.3 L, wait, measure, then top up slowly. Do not exceed MAX on the dipstick.

## Coolant

| Field | Value |
|---|---|
| Danish | Kølervæske |
| English | Coolant / antifreeze |
| Buy amount | 5 L ready-mixed is normally enough for a normal drain/refill; a full flush may need more |
| Type | PSA-compatible OAT coolant, ready-mixed or concentrate mixed 50/50 with demineralised water |
| Recommended | TotalEnergies / Febi / Mannol / Comma that states Peugeot/Citroen compatibility |
| Status | Exact factory spec not yet verified |

Note: coolant is for engine cooling. It does not make the aircondition colder.
