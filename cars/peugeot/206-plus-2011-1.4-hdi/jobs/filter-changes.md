# Job: Filter changes

## Air filter

| Field | Value |
|---|---|
| Danish | Luftfilter |
| English | Air filter |
| OEM | 1444 TV |
| Recommended brands | Purflux, MANN, Bosch, Mahle, Hengst |
| Difficulty | Easy |

Notes:

- Autodoc can show more than one filter size for the same model.
- Do not order by size alone.
- Search by OEM `1444 TV` and compare with the old filter if in doubt.

## Cabin / pollen filter

| Field | Value |
|---|---|
| Danish | Kabinefilter / pollenfilter |
| English | Cabin filter / pollen filter / passenger compartment particle filter |
| OEM | 6447 AZ |
| Extra reference | VCF127 |
| Recommended brands | MANN, Bosch, Purflux, Mahle, Hengst |
| Preferred type | Activated carbon |
| Difficulty | Easy/medium depending on access |

Notes:

- A blocked cabin filter can reduce airflow from the vents.
- It will not fix low refrigerant, but it can make A/C feel weak.

## Fuel filter cartridge

| Field | Value |
|---|---|
| Danish | Brændstoffilter-indsats |
| English | Fuel filter cartridge |
| OEM | 1906 E6 |
| Complete assembly | 1901 97 |
| Recommended brands | Purflux, MANN, Bosch, Mahle, Hengst |
| Difficulty | Medium |

Notes:

- This is important on a diesel because dirt/water can damage the injection system.
- Be careful not to introduce dirt into the fuel system.
- After changing, prime/bleed according to the correct procedure.
